CUTEst tutorial

This tutorial shows how to solve a problem from CUTEst.jl with ExactPenalty.jl.

Equality-Constrained Problems

ExactPenalty.jl solves problems of the form minimize f(x) s.t. c(x) = 0. Therefore, choose a CUTEst problem containing only equality constraints.

1. Load a CUTEst problem

In this example, we use the Hock–Schittkowski problem HS6.

using CUTEst

nlp = CUTEstModel("HS6")
 Downloading artifact: sifcollection

2. Solve with ExactPenalty

using ExactPenalty

stats = L2Penalty(nlp; print_level = 1)
