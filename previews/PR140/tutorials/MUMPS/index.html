MUMPS tutorial

This tutorial shows how to use the MUMPS linear solver to solve a problem from the CUTEst.jl (see this tutorial) collection with ExactPenalty.jl.

Extensions

ExactPenalty.jl uses an extension to load the MUMPS linear solver. Therefore, you need to load both MPI.jl and MUMPS.jl. Our algorithm will throw a warning and switch to the default LDLFactorizations.jl package if you try to use MUMPS without loading the required packages.

1. Load a CUTEst problem

In this example, we choose a medium size problem MSS1.

using CUTEst

nlp = CUTEstModel("MSS1")

2. Load the MUMPS packages

using MPI, MUMPS

3. Solve with ExactPenalty

using ExactPenalty

stats = L2Penalty(nlp; linear_solver = "mumps", print_level = 1)

4. Finalize the CUTEst model

Once the CUTEst problem has been used, you should finalize it, see the CUTEst documentation.

finalize(nlp)
How To Check?

You can verify that MUMPS is correctly being used by inspecting the output of the solver with the option print_level > 1.
